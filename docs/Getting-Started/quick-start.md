# Quick Start

Follow the steps below to start the environment.

---

# Clone Repository

```bash
git clone https://github.com/your-repository/project.git
```

---

# Install Dependencies

```bash
npm install
```

---

# Start Development Server

```bash
npm run start
```

---

# Open Browser

Visit:

```text
http://localhost:3000
```

---

# Build Production Version

```bash
npm run build
```

---

# Troubleshooting

## Port Already In Use

```bash
npx kill-port 3000
```
